# Branch Naming

## User stories
- `feature/<ID>-<short-representative-slug>`
- `improvement/<short-slug>` (user-facing improvements to previous implementations)

## Technical non-user-facing work
- `refactor/<short-slug>`
- `chore/<short-slug>`
- `ci/<short-slug>`
- `docs/<short-slug>`
- `fix/<short-slug>`
